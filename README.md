# FHIR Patient Management

This project is a simple patient management interface that
retrieves patient data from a mock FHIR server and display a list of patients
and the patient's details.
this Project focuses on clean code and maintainable code
styling is kept manimal.

---

## Tech Stack

- React
- TypeScript
- Axios
- FHIR (habi fhir R4)

---

## Features

- Fetch patients from a mock FHIR server
- Display a list of patients
- View patient details
- Navigate between patient list and patient detail views
- Basic error handling

---

## How to Run the Project?

1. Clone the repository:
   git clone https://github.com/leen165/fhir-patient-management.git

2. Install dependencies:
   npm install

3. Start the development server:
   npm start

4.The application uses the public FHIR server:
htttps://hapi.fhir.org/baseR4
